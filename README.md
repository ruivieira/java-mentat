# java-mentat

Java AI/ML/DS library.

## Topics

### Explainability

- Counterfactually fair models
- Fairness metrics (TPR)

## Graph theory

- Erdős–Rényi model

## Metrics

- Receiver operating characteristic (ROC)

## Usage

Add the repository

```xml
<repositories>
  <repository>
    <id>github</id>
    <url>https://raw.githubusercontent.com/ruivieira/java-mentat/releases</url>
    <snapshots>
      <enabled>true</enabled>
      <updatePolicy>always</updatePolicy>
    </snapshots>
  </repository>
</repositories>
```
and the dependency

```xml
<dependency>
  <groupId>dev.ruivieira</groupId>
  <artifactId>java-mentat</artifactId>
  <version>1.0.0-SNAPSHOT</version>
</dependency>
```

to your project.
